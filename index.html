 dendygames.kmx.ru - начало моего пути в веб-разработке. Простейший wap-сайт, созданный в далеком 2010 году с помощью конструктора wap.kmx.ru непосредственно с мобильного телефона. Представляет собой набор страниц, связанных гиперссылками, а также эмуляторов и "ромов" 8-битной приставки "Dendy".
 Работая над этим сайтом, я получил первое представление о структуре html-документов, базовых html-тегах и css-свойствах.
 В данный момент сайт недоступен по своему адресу в связи с закрытием проекта "kmx.ru".

 НижРэпQuiz - это некий аналог игры "Кто хочет стать миллионером" для ценителей Нижегородского хип-хопа.
 Некоммерческая история. Дабы не заморачиваться над изящными интерфейсными решениями в сжатые сроки, основополагающим был выбран фреймворк Bootstrap. В данном проекте я впервые поработал с VK API (приложение парсит данные записей со стены публичной страницы).

 HTML Visual Editor - визуальный конструктор html-страниц, по совместительству - мой дипломный проект, вся логика которого обрабатывается на стороне клиента (в т.ч. непосредственно в браузере генерируется и итоговый html-файл).
 Имеет ряд недороботок и недостатков, однако в процессе написания был получен бесценный опыт использования множества полезных инструментов (jQuery, Bootstrap, FontAwesome, Gulp, PostCSS, SASS).
